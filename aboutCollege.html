📄 About This Project

The Centralized College Database is a DBMS project built for SRM (Sem 4). It models a college with multiple academic departments offering courses, each taught by a single instructor, with students enrolled across courses.

Tech Stack

  • 🟢 Node.js + Express
  • 🐬 MySQL (mysql2)
  • 🎨 Vanilla CSS (Inter font)
  • 📊 D3.js (tree viz)
  • ✨ Canvas particles

Database Schema

  • 📦 department (id, name)
  • 👨‍🏫 instructor (id, name, dept_id)
  • 📖 courses (id, name, dept_id, inst_id, duration)
  • 🎓 student (id, course_id, name, year, age, addr)
